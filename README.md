# Password Audit Tools

A collection of lightweight PoCs to support the concepts discussed in the Password Audit article series:
1. [Password Audits Part 1: NTDS Extraction](https://mollysec.com/posts/password-audits-part-1/)
2. [Password Audits Part 2: Hash Organisation](https://mollysec.com/posts/password-audits-part-2/)
3. [Password Audits Part 3: Cracking Hashes](https://mollysec.com/posts/password-audits-part-3/)
4. [Password Audits Part 4: Analysing Results](https://mollysec.com/posts/password-audits-part-4/)

---
## Tools

### hash-organiser

A Bash-based tool for processing NTDS dumps and preparing datasets for cracking and analysis. See: [`hash-organiser`](https://github.com/CSpanias/pass-audit-tools/tree/main/hash-organiser).

---

### phase-cracker

A Python-based tool for estimating and executing phased Hashcat attacks. See: [`phase-cracker`](https://github.com/CSpanias/pass-audit-tools/tree/main/phase-cracker).
